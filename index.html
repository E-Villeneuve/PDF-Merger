About

This is a lightweight website designed to allow users to combine multiple PDFs into a single file, without the requirement to upload to some dubious server with questionable privacy protocols. Everything is handled in-browser, with nothing stored on the server side.

If still skeptical, disconnect from the Internet while on this page; the merger will remain perfectly functional!
